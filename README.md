# 💰 PriceDrop AI

![Python](https://img.shields.io/badge/Python-3.10+-blue) ![Streamlit](https://img.shields.io/badge/Streamlit-1.32+-red)

**AI-powered price tracker and deal predictor.** Enter a product name, simulate realistic price history, and get AI-powered buy/wait recommendations.

## Features
- 📈 Simulated price history with realistic patterns (sales, inflation, seasonality)
- 🤖 AI buy/wait/watch recommendation
- 📉 Price trend analysis (rising/falling/stable)
- 🎯 Predicted best price window
- 🔔 Deal score out of 10
- 📦 Works for any product category

## Run
```bash
pip install -r requirements.txt
GEMINI_API_KEY=your_key streamlit run app.py
```
